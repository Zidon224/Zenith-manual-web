Scripted Render


With this module you can create resource packs using C# scripts to make everything u want(If you don't know how to create C# scripts for your resource pack you can use the textured module render)


Here is the resource pack list

If you're adding a new resource pack folder or ZRP click on the reload list to show it to the list

If you're modifying the resource pack folder you can refresh the pack by clicking on reload script

Here you can set the render note speed render and the keyboard range


Textured module


With this module you can create much more easy a resource pack using json scripts

The packs list is similar to the Scripted render

To reload a new pack added click on the reload list button and the resource pack will be displaied to the list


And the misc settings are common to each module
